# Чек-лист оцінювання ефективності лектора / інструктора

Інтерактивний PWA веб-додаток для методологічного оцінювання викладів, збереження результатів та друку звітів.

---

## 📌 Основні можливості

- **100-бальна система оцінювання**: 5 ключових категорій з гнучкими чекбоксами та числовими інпутами.
- **Шкала конверсії оцінок**:
  - `95 – 100 балів` ➔ **5** (Відмінно)
  - `90 – 94 балів` ➔ **5-**
  - `85 – 89 балів` ➔ **4+**
  - `75 – 84 балів` ➔ **4** (Добре / Зараховано)
  - `68 – 74 балів` ➔ **3+**
  - `60 – 67 балів` ➔ **3** (Задовільно)
  - `< 60 балів` ➔ **2** (Незадовільно)
- **Друк та PDF**: Оптимізована 1-сторінкова друкована форма А4 (Акт оцінювання).
- **Локальна історія**: Автоматичне збереження результатів у браузері (`localStorage`) + експорт/імпорт у JSON.

---

## 📊 Структура критеріїв оцінювання (100 балів)

1. **I. Організація (15 балів)**
   - Привітання, представлення, озвучення теми, мети та правил (5 балів)
   - Наявність логічного плану та чіткої структури (5 балів)
   - Змістовні підсумки наприкінці виступу (5 балів)
2. **II. Матеріал / Експертність (30 балів)**
   - Подача «своїми словами» без читання з аркуша (5 балів)
   - Зрозумілість пояснення (5 балів)
   - Точність інформації, відсутність фактологічних помилок (10 балів)
   - Практична цінність, приклади з досвіду (5 балів)
   - Впевнена та аргументована відповідь на запитання (5 балів)
3. **III. Невербаліка та Ораторство (20 балів)**
   - Динамічний голос, відсутність монотонності (5 балів)
   - Чітка дикція, відсутність слів-паразитів (5 балів)
   - Постійний зоровий контакт з аудиторією (5 балів)
   - Відкрита поза, доречна жестикуляція (5 балів)
4. **IV. Взаємодія з аудиторією (20 балів)**
   - Активізація уваги, запитання слухачам (10 балів)
   - Самостійне управління дисципліною та увагою (10 балів)
5. **V. Наочність та Таймінг (15 балів)**
   - Якісна візуалізація, робота з презентацією (10 балів)
   - Дотримання тайм-менеджменту / регламенту (5 балів)

---

## 🛠 Використання
1. Заповніть відомості (Псевдо лектора, Тема, Методолог, Дата).
2. Проставте бали/чекбокси за виступ.
3. Натисніть **«Зберегти»** (для збереження у пам'яті пристрою) або **«Друк / PDF»** для формування Акта оцінювання.
