# Чек-лист оцінювання ефективності лектора / інструктора

Інтерактивний PWA веб-додаток для методологічного оцінювання викладів, збереження результатів та автоматичної синхронізації з **Google Таблицями** на Google Диску.

---

## 📌 Основні можливості

- **100-бальна система оцінювання**: 5 ключових категорій з гнучкими чекбоксами та числовими інпутами.
- **Шкала конверсії оцінок**:
  - `95 – 100 балів` ➔ **5** (Відмінно)
  - `90 – 94 балів` ➔ **5-**
  - `85 – 89 балів` ➔ **4+**
  - `75 – 84 балів` ➔ **4** (Добре / Зараховано)
  - `68 – 74 балів` ➔ **3+**
  - `60 – 67 балів` ➔ **3** (Задовільно)
  - `< 60 балів` ➔ **2** (Незадовільно)
- **Інтеграція з Google Таблицями**: Надсилання оцінювань в один клік у вашу таблицю на Google Диску.
- **Друк та PDF**: Оптимізована 1-сторінкова друкована форма А4 (Акт оцінювання).
- **Локальна історія**: Автоматичне збереження результатів у браузері (`localStorage`) + експорт/імпорт у JSON.

## 📊 Структура критеріїв оцінювання (100 балів)

1. **I. Організація (15 балів)**
   - Привітання, представлення, озвучення теми, мети та правил (5 балів)
   - Наявність логічного плану та чіткої структури (5 балів)
   - Змістовні підсумки наприкінці виступу (5 балів)
2. **II. Матеріал / Експертність (30 балів)**
   - Подача «своїми словами» без читання з аркуша (5 балів)
   - Зрозумілість пояснення (5 балів)
   - Точність інформації, відсутність фактологічних помилок (10 балів)
   - Практична цінність, приклади з досвіду (5 балів)
   - Впевнена та аргументована відповідь на запитання (5 балів)
3. **III. Невербаліка та Ораторство (20 балів)**
   - Динамічний голос, відсутність монотонності (5 балів)
   - Чітка дикція, відсутність слів-паразитів (5 балів)
   - Постійний зоровий контакт з аудиторією (5 балів)
   - Відкрита поза, доречна жестикуляція (5 балів)
4. **IV. Взаємодія з аудиторією (20 балів)**
   - Активізація уваги, запитання слухачам (10 балів)
   - Самостійне управління дисципліною та увагою (10 балів)
5. **V. Наочність та Таймінг (15 балів)**
   - Якісна візуалізація, робота з презентацією (10 балів)
   - Дотримання тайм-менеджменту / регламенту (5 балів)

## 🚀 Покрокова інструкція з підключення Google Таблиці

### Крок 1: Відкрийте Apps Script у Google Таблиці
1. Відкрийте потрібну **Google Таблицю** на своєму Google Диску.
2. Перейдіть у верхньому меню: **`Розширення` (Extensions)** ➔ **`Apps Script`**.

### Крок 2: Вставте код скрипта
Видаліть стандартний код у файлі `Code.gs` та вставте наступний готовий скрипт:

```javascript
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    
    // Отримуємо або створюємо аркуш "Журнал оцінювань"
    var sheet = ss.getSheetByName("Журнал оцінювань") || ss.getActiveSheet();
    
    // Створюємо шапку таблиці, якщо аркуш порожній
    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        "Дата", 
        "Псевдо лектора", 
        "Тема виступу", 
        "Бал (100)", 
        "Оцінка (1-5)", 
        "Статус", 
        "Методолог", 
        "Зауваження та рекомендації"
      ]);
      sheet.getRange(1, 1, 1, 8)
        .setFontWeight("bold")
        .setBackground("#3b82f6")
        .setFontColor("#ffffff");
    }
    
    // Додаємо рядок із даними з чек-листа
    sheet.appendRow([
      data.date || new Date().toLocaleDateString("uk-UA"),
      data.instructor || "Невідомий",
      data.topic || "—",
      data.totalScore || 0,
      data.grade5 || "—",
      data.statusText || "—",
      data.evaluator || "—",
      data.notes || ""
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ result: "success" }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ result: "error", error: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```

### Крок 3: Опублікуйте скрипт як Web App
1. Натисніть синю кнопку **`Розгорнути` (Deploy)** у правому верхньому куті ➔ **`Нове розгортання` (New deployment)**.
2. Натисніть на значок шестерні ⚙️ поруч із *Виберіть тип* та оберіть **`Веб-програма` (Web app)**.
3. Заповніть параметри:
   - **Опис**: *Оцінювання лекторів*
   - **Виконувати від імені**: *Я (ваш email)*
   - **Хто має доступ (Who has access)**: ⚠️ **`Усі` (Anyone)** *(обов'язково)*.
4. Натисніть **`Розгорнути` (Deploy)**.
5. Надайте дозволи (Google запитає дозвіл на доступ до таблиці — оберіть свій акаунт ➔ *Advanced* ➔ *Go to project (unsafe)* ➔ *Allow*).
6. Скопіюйте **URL веб-програми** (вигляду `https://script.google.com/macros/s/.../exec`).

### Крок 4: Вкажіть URL у Чек-листі
1. Відкрийте веб-додаток [index.html](file:///d:/Checklist/index.html).
2. Натисніть кнопку ⚙️ поруч із кнопкою `В Google Таблицю`.
3. Вставте скопійований URL і натисніть **«Зберегти URL»**.

## 🛠 Використання
1. Заповніть відомості (Псевдо лектора, Тема, Методолог, Дата).
2. Проставте бали/чекбокси за виступ.
3. Натисніть 🟢 **`В Google Таблицю`**.
4. Дані миттєво відправляться в Гугл Таблицю на вашому Гугл Диску!
